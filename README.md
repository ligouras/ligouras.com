# ligouras.com

Coming-soon page for [ligouras.com](https://ligouras.com), served via GitHub Pages.

## Stack

Static `index.html` — no build step. Minimal dark (OLED) theme, Archivo + Space Grotesk fonts, inline SVG social icons.

## Deploy

GitHub Pages serves the `main` branch. The `CNAME` file maps the site to the `ligouras.com` apex domain.

To preview locally, open `index.html` in a browser or run:

```bash
python3 -m http.server
```
